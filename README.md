# Projeto MySQL - Análise de Dados

## Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar modelagem, criação e consultas em banco de dados utilizando MySQL.

## Tecnologias utilizadas
- MySQL
- MySQL Workbench
- SQL

## Estrutura do projeto
- Criação do banco de dados
- Criação das tabelas
- Inserção de dados
- Consultas SQL
- Joins
- Agregações
- Subconsultas
- Views
- Procedures/Functions

## Principais consultas realizadas
- Consulta de pedidos por status
- Ranking de produtos mais caros
- Total de pedidos por cliente
- Produtos por categoria
- Clientes sem pedidos
- Total vendido por pedido
- Consultas com JOIN
- Subconsultas
- Criação de views

## Aprendizados
Neste projeto pratiquei conceitos importantes de SQL, como criação de tabelas, relacionamento entre dados, filtros, agrupamentos, joins, views e procedures.
